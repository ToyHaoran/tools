# 久坐提醒

一个极简的 [AutoHotkey v2](https://www.autohotkey.com/) 小工具。

- 运行 `WorkTimer.ahk` 后，`时:分:秒` 会实时显示在任务栏通知区域左侧、紧贴任务栏边缘的位置。
- 双击时间可将计时清零并重新开始。
- 按住计时条任意位置的右键可拖动，将它移到任意位置；位置会自动记住。
- 鼠标位于计时条时点击滚轮键（中键）可打开与托盘图标相同的设置菜单。
- 默认使用紧凑的 100 × 26 尺寸与小号字体。托盘菜单的「显示大小」可选小、标准、大号，或输入自定义的宽度 × 高度 × 字号。
- 背景为透明；文字会根据下方背景自动反色：浅色背景显示黑字，深色背景显示白字。
- 右键系统托盘中的 AutoHotkey 图标可打开全部菜单：重新开始、暂停/继续计时、启动/停止鼠标穿透、启动/停止全屏隐藏、提醒时点、提醒声音、显示大小、恢复自动停靠与退出。启用鼠标穿透后，计时条不再拦截鼠标操作；可从托盘菜单关闭。启用全屏隐藏后，前台全屏窗口出现时计时条会自动隐藏，退出全屏即恢复。暂停时计时冻结且不会发出提醒。
- 默认提醒音为较长的风铃声；托盘的「提醒声音」中可切换风铃、登场音或系统通知音。选择后会立即试听。
- 每次提醒触发后，声音会持续约 30 秒，并每 4 秒重复一次；暂停或重新开始会停止后续重复。
- 每个计时周期仅在设定的第 N 分钟提醒一次；只有双击时间重置后，才会重新安排一次提醒。
- 托盘菜单的「倒计时」支持输入秒数（如 `90`）或分:秒（如 `25:30`）并开始倒数；时间归零会发出系统通知与提示音，暂停/继续也会作用于倒计时。

> Windows 11 不再支持第三方程序稳定地把任意窗口嵌进任务栏内部，因此计时条设计为紧贴任务栏显示，以保证始终可见。

## 使用

1. 安装 AutoHotkey v2。
2. 双击 `WorkTimer.ahk` 运行。
3. 如需开机自动运行：按 `Win + R`，输入 `shell:startup`，将 `WorkTimer.ahk` 的快捷方式放入打开的文件夹。

若通知未显示，请在 Windows「设置 → 系统 → 通知」中允许 AutoHotkey 的通知。
